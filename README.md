
# BG-ProAir Training Package

This repository contains the BG‑ProAir Interactive Training Quiz and the Parts Blank reference PDF.

## Included Files
- **BG_ProAir_Final_Interactive_Quiz_v10_TechNouns_Synonyms.html** — Interactive training quiz.
- **BG-ProAir Parts Blank.pdf** — Diagram pages used for part‑identification questions.
- **README.md** — This file.

## How to Use
1. Ensure the HTML file and PDF remain in the same folder.
2. Open the HTML file in a modern web browser.
3. Click **Start Test** to begin.

## Features
- Statement‑based and parts‑based questions
- Synonym distractors
- Built‑in scoring and feedback
- PDF diagram preview system
- Highlighting and zoom tools
